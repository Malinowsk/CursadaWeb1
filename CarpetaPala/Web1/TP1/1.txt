La estructura de un documento HTML esta dividida en 2 partes:
El HEAD en el cual se define el 
	<title>: el titulo del documento, es obligatorio
	<meta>: fuerza a la pagina a realizar ciertas funciones
	<link>: links de referencia a archivos externos o paginas de internet

El BODY en el cual se define la parte visual del documento, en esta seccion se aplica la estructura general de la pagina web.

2_
Los estilos sirven para poder personalizar una pagina web, dar color, forma, tamaño, acomodar los elementos de la pagina web segun el diseño que se tenga pensado para esta.
Los estilos se aplican cargando un archivo "archivo.css" al documento "documento.html" mediante la etiqueta <link reel="stylesheet" href="archivo.css"> en la parte del HEAD, el "href" hace referencia a la ubicacion del archivo "archivo.css" dentro del directorio donde esta el documento HTML.

3_
Existen 3 tipos de selectores:
	*Los selectores de Tipo: conviene usarlos cuando se desea aplicar un estilo general a todas las etiquetas de un tipo determiando dentro de un documento HTML
	*Los selectores de Clase: para estos selectores hay que definir una clase dentro de las etiquetas HTML, se utilizan para poder personalizar una clase de elemento, por ejemplo un buscador o un carrucel de imagenes.
	*Los selectores por ID: estos selectores no son recomendados, para utilizarlos hay que aplciar un ID a una etiqueta especifica del documento HTML.
